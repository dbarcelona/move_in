# Lakeside Move-In Tracker

A web-based tool to help manage moving tasks and plan your new space in Oakland's Lakeside neighborhood.

## Features

### 📋 To-Do List
- Add tasks with categories (Packing, Utilities, Cleaning, Moving, Shopping, Other)
- Set due dates for tasks
- Mark tasks as complete
- Filter tasks by category
- Automatic sorting (incomplete tasks first, then by due date)
- Data persists in your browser

### 🏡 Space Planner
- Drag and drop furniture items to visualize your layout
- Available items: Bed, Sofa, Table, Desk, Shelf
- Layout is saved automatically
- Clear and start over anytime

## How to Use

### Opening the App
1. Open the project folder
2. Double-click `index.html` to open in any web browser
3. That's it! No server needed.

### Managing Tasks
1. Enter your task in the text box
2. Select a category
3. Optionally set a due date
4. Click "Add Task" or press Enter
5. Check the box when complete
6. Click "Delete" to remove

### Planning Your Space
1. Click the "Space Planner" tab
2. Click a furniture button to add it to the room
3. Drag items around to plan your layout
4. Your layout saves automatically

## Technical Details

**Technologies Used:**
- HTML5 - Structure
- CSS3 - Styling with gradients and animations
- Vanilla JavaScript - Interactivity (no frameworks!)
- localStorage API - Data persistence

**Browser Compatibility:**
Works in all modern browsers (Chrome, Firefox, Safari, Edge)
